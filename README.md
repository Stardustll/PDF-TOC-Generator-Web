# PDF 目录自动生成器 (PDF-TOC-Generator-Web)

> 纯 vibe coding 编写的自用玩具，自行测试体验

基于 Python Flask 开发的 Web 应用，通过 **AI 视觉大模型** 自动识别 PDF 文档的章节目录，并将其生成为 PDF 标准书签，方便快速导航。

## 🌟 主要功能

- **PDF 上传与预览**：支持本地 PDF 文件上传，在网页端实时预览页面内容。
- **AI 视觉识别**：调用 OpenAI 兼容的 Vision API（GPT-4o、Qwen-VL 等）对目录页截图进行智能识别，准确率高。
- **在线目录编辑**：交互式编辑器，支持修改标题、页码、调整层级、上下移动和删除条目。
- **页码偏移校正**：一键设置全局偏移量，自动修正扫描版 PDF 的页码差值。
- **书签一键生成**：将编辑好的目录写回 PDF 文件，生成标准 PDF 书签并下载。
- **管理员后台**：独立的受密码保护的管理界面，统一配置全局 AI 服务信息。


## 🚀 快速启动

### 环境准备

Python **3.10+**（已在 3.12、3.13 环境下通过测试）。

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置管理员密码（可选）

默认管理员密码为 `admin123`，**强烈建议在开放部署前通过环境变量修改**：

```bash
# Linux / macOS
export ADMIN_PASSWORD=your_strong_password

# Windows PowerShell
$env:ADMIN_PASSWORD = "your_strong_password"
```

### 运行应用

```bash
python app.py
```

| 地址 | 说明 |
| :--- | :--- |
| `http://127.0.0.1:5000/` | 主界面（用户使用） |
| `http://127.0.0.1:5000/admin` | 管理后台（配置全局 AI 服务） |

## ⚙️ 管理员配置

首次部署后，访问 `/admin` 页面登录，填入 AI 服务的 **API Base URL** 和 **API Key**，保存后全站生效。所有用户无需自行填写即可直接使用 AI 识别功能。

普通用户也可以在主界面的「AI 配置」弹窗中**临时覆盖**当前任务的 API 信息，该修改仅对本次处理有效，会话结束后自动清除，不影响全局设置。

## 📝 使用流程

1. **上传 PDF**：在主页选择本地 PDF 文件上传。
2. **标记目录页**：在左侧页面列表中双击 / Ctrl+点击选中目录所在页。
3. **识别目录**：
   - 点击「AI 识别」调用视觉大模型自动提取目录（推荐）。
   - 点击「提取文本」使用本地提取（无 API 时使用，未测试）。
4. **编辑校对**：在右侧编辑器中修正层级、标题、页码及偏移量。
5. **生成下载**：点击「保存书签」，下载带有标准书签的 PDF 文件。

## ⚠️ 注意事项

- 单文件最大上传限制为 **200MB**，可在 `config.py` 中修改 `MAX_CONTENT_LENGTH`。
- AI 视觉识别需要有效的 API Key，请在管理后台配置。
- OCR 准确度受文档图像质量及排版复杂度影响。
- 本项目使用 Flask 内置开发服务器，**不建议直接用于生产环境**，建议配合 Gunicorn + Nginx 部署。
